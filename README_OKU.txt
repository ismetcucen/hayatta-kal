ÖNEMLİ: PROJEYİ ÇALIŞTIRMA TALİMATI

Yaşanan teknik aksaklıklar için özür dilerim. Projenizin tüm kodları (81 il, harita, oyun) bu klasörde hazır.

Otomatik kurulum sürekli iptal edildiği için, bunu elinizle tek bir seferde yapmanız en garantisidir.

1. Bilgisayarınızda "Terminal" uygulamasını açın.
2. Aşağıdaki komutu kopyalayıp Terminal'e yapıştırın ve Enter'a basın:

cd /Users/ismetcucen/.gemini/antigravity/scratch/hayatta_kal && npm install && npm run build

3. Bu işlem bitince bu klasörde "dist" diye bir klasör oluşacak.
4. O klasörü Netlify Drop sitesine sürükleyip paylaşabilirsiniz.

Alternatif (Bilgisayarda Oynamak İçin):
cd /Users/ismetcucen/.gemini/antigravity/scratch/hayatta_kal && npm install && npm run dev
